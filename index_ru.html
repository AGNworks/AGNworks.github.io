AI Security

In this intership project we created a neural network, which is detecting a weapon with high precision on a frame from a security camera. We created this project with the help of YOLOv8 architecture. And with the help of NestJS we created an API server for the implementation of this mode.

Work schedule with AI

With this development we can make a smart camera from the already installed security cameras in an office or manufacturing space. The working process of this system is the following: the model get a frame from the camera, if there are people on the frame who were not detecting before on that day the system add this information to a table as arriving time, at the end of the workday if an employee is no more detected then the time of his last appirance will be added as time of leaving. This project is not finished yet, but to get more information visit my github.

More projects

M

I made the following projects at my workplaces or as a hobby project. All of them are done with the help of programming and engineering in cooparation.

DMX-512 controller on the base of Raspberry Pi

This gadget I created to test easier and faster the lightning systems, what we are working on DMX-512 protocol. The case I draw in Fusion360 and printed the elements on 3d printer from PETG plastic. The system works with the help of a Raspberry Pi, with Raspbian installed on it, and an arduino nano, which is transfering the signal with the help of RS485 modul. For more information about this project visit the github repository.

Wireless bottle for theatre performance

This interesting bottle we created for a theatre performance, where it was needed to be opened a champagne bottle without human help. When the button on the remote is pressed a valve is opening and with the help of the compressed air the cork fly out of the bottle. The bottle is printed on 3D printer, the code I wrote in C++ for the bottle and for the remote too.

DMX-512 controller on Nodemcu

This is a more compact version of a DMX-512 controller, working from battaries 18650, and creating a hostpot for the controlling interface. We can connect with our phone and open the correct ip address. The code I wrote in C++ , it is running on a NodeMCU microcontroller. If you are intereted in this project visit my github.

50th Birthday Party

This project I made with collages at work. The number 50 is from segments hanging on different rods. All of the rods are connected with timing belt, it makes 360grad than stops every time when the number is in correct position. In this project I used some sensors to check the position of the rods. The brain of this project a microcontroller, programed by me in C++.

Filament dryer for 3D printing

My friend asked me to create a controlling system for his DIY filament dryer. So I made a self controlling heating system with the availability setting up the process parameters. Here I used a temperature sensor, arduino nano, oled display and an encoder for navigating in the menu. Also written in C++. Link to the github project.

Interactive Busyboard - Moscow Urban Forum 2022

In this project I created the system, which implements the idea of the designer. This is a game for children, where they can move on the field a puppet. At different stages happens different effects (lights up windows in buildings etc.). Also made with the help of microcontroller, SPI LED strips and some sensors.

Foot keyboard

I created this keyboard for my engineer friend to work faster in AutoCad. With this you can use duplicate 3 buttons from the keyboard to press them with the help of you leg. Link to the github project.

Frozen clock for theatre performace

All the wheels are turning with different speeds in this decoration and it has a remote controlled start system. I designed the electrical and the electro-mechanical part of this project. Controlled and cooparated in the assembly process.

LED flowers for dancing show in Doha 2020

We wreated 52 pcs of wireless LED flowers for dancing show. It is supplied from a LiPo battary and controlled wirelessly via DMX-512 protocol. I designed the electrical part of the flowers and participated in the assembly process. I traveled with them to Katar to give tech support in case of need.

Theatre performace -Laurel-

It is a huge decoration with a lots of lighting feauters all of them are working on the protocol with DMX-512. I designed and controlled the assmebly process of the electrical part of this project. I created a controlling system for the 26 electric blinds on the front side of the construction.
