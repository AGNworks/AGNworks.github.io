Охранная система с помощью искусственного интеллекта

В рамках стажировочного проекта мы создали нейронную сеть, которая с высокой точностью обнаруживает оружие на кадре с камеры видеонаблюдения. Мы создали этот проект с помощью архитектуры YOLOv8, и интегрировали в API-сервер для заказчика. Стек: OpenCV, Yolo, NestJS, Docker

Ведение графика работы сотрудников с AI

С помощью этой разработки мы можем создать интеллектуальную камеру из уже установленных камер видеонаблюдения в офисе или производственном помещении. Рабочий процесс этой системы заключается в следующем: модель получает кадр с камеры, если в кадре есть люди, которые ранее не были обнаружены в этот день, система добавляет эту информацию в таблицу как время прибытия, в конце рабочего дня, если сотрудник больше не обнаружен, то время его последнего появления будет добавлено, как конец его рабочего дня. Этот проект еще не завершен, текушую версию можете посмотреть на моем гитхубе.

Контроллер DMX-512 на базе Raspberry Pi

Этот гаджет я создал для того, чтобы проще и быстрее тестировать системы освещение, работающие по протоколу DMX512. Система работает на Raspberry Pi, на котором установлен Raspbian, и Arduino nano, который передает сигнал с помощью модуля RS485. Для получения дополнительной информации об этом проекте посетите сайт гитхуб.

Беспроводная бутылка для театрального представления

Эту интересную бутылку мы создали для театрального представления, где нужно было открыть бутылку шампанского без помощи человека. Когда кнопку нажат на пульте дистанционного управления открывается клапан, и с помощью сжатого воздуха пробка вылетает из бутылки. Бутылка напечатана на 3D-принтере, код, я написал в C++ для бутылки и для пульта дистанционного управления тоже.

Контроллер DMX-512 на Nodemcu

Это более компактная версия контроллера DMX512, работающая от батареек 18650 и создающая беспроводную точку для доступа к интерфейсу управления. Я написал код на C++, запущен на микроконтроллере NodeMCU. Если проект Вам интересен, посетите мой гитхуб.

Юбилей - 50

Число 50 состоит из сегментов, подвешенных на разных валах. Все валы соединены зубчатым ремнем через швиф. Конструкция поворачивается на 360 градусов и останавливается каждый раз, когда сегменты находятся в правильном положении. В этом проекте я использовал несколько датчиков для проверки положения валы. Мозгом этого проекта является микроконтроллер, запрограммированный мной на C++.

Здесь я использовал датчик температуры, arduino nano, oled-дисплей и энкодер для навигации по меню. Ссылка на проект тут.
